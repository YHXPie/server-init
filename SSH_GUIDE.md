================================================================
            如何生成 SSH 密钥 (How to generate SSH Key)
================================================================

## [选项 1] Windows 10/11 用户
----------------------------------------------------------------
1. 按下 "Win徽标键 + X" 组合键，选择 "Windows PowerShell" 或 "终端"。
2. 在弹出的蓝色窗口中，粘贴并运行以下命令：
   ssh-keygen -t ed25519 -C "my_server_key"
3. 建议为密钥创建一个密码
4. 运行以下命令查看并在屏幕上“选中并复制”公钥内容：
   cat ~/.ssh/id_ed25519.pub

## [选项 2] macOS / Linux 用户
----------------------------------------------------------------
1. 打开 "终端" (Terminal) 应用。
2. 粘贴并运行以下命令：
   ssh-keygen -t ed25519 -C "my_server_key"
3. 建议为密钥创建一个密码
4. 运行以下命令查看并在屏幕上“选中并复制”公钥内容：
   cat ~/.ssh/id_ed25519.pub

## [选项 3] 使用旧版 Windows / PuTTY
----------------------------------------------------------------
1. 打开 PuTTYgen 软件。
2. 点击 "Generate" 按钮，并在空白处移动鼠标生成随机数。
3. 复制上方 "Public key for pasting..." 下框中的所有内容。
   (以 ssh-rsa ... 开头)

================================================================
获取到公钥内容后（以 ssh-xxxx 开头的一长串字符），
请运行 sudo bash init2.sh 并将其粘贴进去
